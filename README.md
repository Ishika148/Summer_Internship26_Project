# Tech News Research Agent

## Overview

Tech News Research Agent is an AI-powered automation workflow built using n8n. The workflow collects the latest technology news from an RSS feed, analyzes the articles using OpenAI GPT-5 Mini, generates a professional technology intelligence report, and automatically emails the report to recipients.

## Features

* Automated daily execution using Schedule Trigger
* Fetches technology news from RSS feeds
* AI-powered article analysis and summarization
* Generates executive-style technology reports
* Sends reports via Gmail automatically
